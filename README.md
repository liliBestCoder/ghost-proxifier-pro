# Ghost Proxifier Pro (幽灵代理器 专业版)

Ghost Proxifier Pro 是基于开源版本深度定制的**进阶高性能代理工具**。它不仅继承了开源版的高效内核，更在用户体验、自动化处理与核心拦截算法上进行了全面进化。

> [!TIP]
> **Pro 版 vs 开源版**：如果您追求更极致的视觉体验、更智能的自动化接管以及更稳定的底层驱动，Pro 版是您的不二之选。

---

## 🔥 Pro 版 核心优势 (Premium Features)

### 💎 现代化酷炫 UI 界面
Pro 版采用了全新的桌面 UI 设计规范，为您带来**玻璃拟态 (Glassmorphism)** 与**动态暗黑模式**的极致视觉享受。
- **动态仪表盘**：实时动画展示网络拓扑与流量峰值。
- **微交互设计**：每一次点击与状态切换都伴随着平滑的过渡动画，操作不再枯燥。

### 📡 智能探测注入 (Smart Injection Detection)
这是 Pro 版最人性化的功能改进，让您告别复杂的配置：
- **“全家桶”自动接管**：您只需要选中目标程序（如某个游戏或浏览器），Pro 版会自动识别并追踪该程序启动的所有关联进程。这意味着您不再需要分析复杂的父子进程结构，一次点击，全量接管。
- **即启即用，无感运行**：软件会自动感应新程序的开启并毫秒级完成接入。只要代理开关打开，剩下的工作全部由 Pro 引擎在后台静默完成，真正做到“零干预”使用体验。
- **高级兼容性支持 (ConnectEx)**：针对现代 Windows 应用（如 Xshell、UWP 应用等）的特殊连接模式提供原生支持，网络穿透力更强。
- **状态自动维护**：内置稳定性看门狗，当系统网络波动导致配置失效时，软件会自动尝试拉回代理状态，确保连接不中断。
